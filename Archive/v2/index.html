Infographer

An easy to use infographic creator

A RESTful MEAN stack web application that lets users easily create, save, print and share infographics. Use one of the available backgrounds, text, shapes and images from Flickr or upload your own and customize. Supports Google, Facebook and email sign up with password recovery using Twilio SMS. Save infographics to your Dropbox or computer, share on Facebook or just print it and stick it on the notice board

Eventor

Find events better

A web app developed using HTML, CSS and JS providing a simple map view with filters showing best matched events from various event sites around the user or their preferred location.
Developed during HackNew 2017 and offered acquisition by two different sponsors
